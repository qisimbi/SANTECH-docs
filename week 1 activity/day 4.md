Today we created the project where we worked on an RFID-based access control system using Arduino, where different RFID cards control a servo motor and display messages on an LCD. 

it included the following components :
Arduino Uno: The main controller that processes data and controls all components
RFID-RC522 module:Reads the unique ID (UID) from RFID cards
servo motor: Acts as a door mechanism (opens, partially opens, or stays closed)
16x2 LCD (I2C): Displays messages like “Access Granted” or “Access Denied”
breadboard:Used to connect all components together
jumper wires :Used to connect all components together
RFID cards:Used to identify different users
