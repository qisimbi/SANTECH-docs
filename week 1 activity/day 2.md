Today was the second day of our internship. The mentor asked us to set up the necessary programming tools on our computers, including Arduino IDE, VS Code, Python, Code::Blocks, and a few others.

After everyone checked that they had the required tools installed, those who were ready were given a small project: to make an LED light up using Arduino. It was a simple hands-on task, but it helped us get familiar with the development environment and understand the basic workflow of programming hardware.

This exercise was a great way to start practical learning seeing how code interacts with real hardware made the concepts we’ve studied feel much more real. It also encouraged teamwork, as we helped each other troubleshoot installation issues and initial coding challenges.
