Today we created the project where we worked on the start security control 
it included the following components :

- Servormotor: For opening the door if the access is granted. 
- Buzzer: For sound alert if the password is wrong.
- Keypad: Provides an interface for typing passwords.
- LCD: For displaying the typing password.
