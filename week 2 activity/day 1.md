Today we studied about Php which means Hypertext Preprocessor: A language that prepares web pages on the server before users see them.

Why we use PHP in real life

- It powers a huge part of the internet.
- It connects websites to databases.
- It runs on the server (not your device).
- It makes websites dynamic.

Why we use php in iot

- It handles data from IoT devices.
- Building dashboards and control panels.
- Connecting IoT systems to websites.
- Storing and managing IoT data.

Real-life example:

Think of a smart home system:

* Sensors collect data (temperature, motion)
* PHP backend receives and stores it
* A web app shows you the data and lets you control devices

- PHP is not used inside the device itself (like on a microcontroller such as Arduino).
